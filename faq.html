❓ Bunny Bot — FAQ

Frequently Asked Questions

General

How do I invite Bunny Bot to my server?

You can invite Bunny Bot using the official invite link provided in our support server or documentation. You’ll need the Manage Server permission.

Is Bunny Bot free?

Yes, Bunny Bot is free to use. Some optional premium features may be added in the future to help with hosting costs.

Why does the bot need so many permissions?

Bunny Bot manages moderation, logging, automod, starboard, bump reminders, and more. Permissions are only requested if required for features like managing roles, deleting messages, or logging events.

Does Bunny Bot support multiple languages?

Right now Bunny Bot defaults to English. Multi-language support is planned, with server owners being able to choose the bot’s language.

Commands & Features

How can I see all commands?

Use ,help to see Bunny Bot’s interactive help menu. Commands are organized by category (Moderation, Utility, Economy, Fun, etc.).

Can I change the prefix?

Currently, the default prefix is ,. Prefix customization will be available soon.

What is AntiNuke and why do I need it?

AntiNuke protects your server from malicious raids by limiting suspicious actions (mass bans, role deletions, channel nukes, etc.). It’s highly recommended for server security.

What is AutoMod?

AutoMod automatically filters unwanted content such as links, invites, spam, mass mentions, and more. You can customize punishments for violations.

How does the bump reminder work?

Bump reminder thanks the user who bumps and reminds them again after 2 hours. It helps keep your server visible on Disboard automatically.

What is the Starboard?

The Starboard highlights popular or funny messages in a special channel when they receive enough ⭐ reactions. It’s a great way to save memorable moments.

Support

How can I report a bug or suggest a feature?

You can report bugs or suggest features in our support server on Discord. Links are provided in the bot’s ,help menu.

How do I contact the developer?

Join our support server or open a GitHub issue on the official repository. Direct contact information is available in the support server.
