Frequently Asked Questions

Answers for setup, permissions, commands, and support flow.

How do I invite Bunny Bot?

Use the invite button on the home page. You need Manage Server permission in the target server.

What prefix does the bot use?

The default prefix is ,. Use ,help or the docs page to discover commands quickly.

Why does Bunny Bot ask for many permissions?

Permissions map directly to enabled features (moderation, logs, tickets, automod, role tools). Disable modules you do not need.

How do I report bugs or request features?

Join the support server and post in the appropriate channel with server ID, command used, and screenshots/log details.

Are confessions anonymous to server members?

Yes for members. Depending on your moderation settings, staff may be able to review submission metadata for abuse handling.

Can I use only a few modules?

Yes. Bunny Bot is modular: configure only what you need and disable command groups you don't use.
