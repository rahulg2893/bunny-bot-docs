🔍 Search Commands

⚒️ Moderation

Moderation keeps your server safe. Jail and mute roles are automated for convenience, no manual setup required.
  • ,kick <user> [reason] — Kick a member from the server.
  • ,ban <user> [reason] — Permanently ban a member.
  • ,unban <user> — Unban a previously banned user.
  • ,warn <user> [reason] — Issue a warning.
  • ,warnings <user> — Check a user’s warnings.
  • ,clearwarns <user> — Clear all warnings.
  • ,timeout <user> <duration> — Temporarily mute.
  • ,jail <user> — Restrict a user to a jail channel.
  • ,unjail <user> — Release from jail.
  • ,lock / ,unlock — Lock or unlock a channel.

🛡️ AutoMod & AntiNuke

Why it matters: AutoMod prevents spam & raids. AntiNuke protects against mass bans, role wipes, or channel nukes.
  • ,am enable — Enable AutoMod.
  • ,am disable — Disable AutoMod.
  • ,am blacklist add <word> — Block bad words.
  • ,am spam <seconds> <messages> — Spam filter.
  • ,am punishment <action> — Choose warn/mute/ban.
  • ,an enable — Enable AntiNuke.
  • ,an threshold <action> <limit> — Limit actions (e.g. bans/minute).
  • ,an defaultpun ban — Default punishment on trigger.

📊 Stats & Invites

Stats give insight into server activity. Invite tracking helps prevent fake/alt raids.
  • ,stats — Today’s stats (messages, joins, leaves, voice).
  • ,stats week — Last 7 days activity chart.
  • ,mstats — Member join/leave stats (30 days).
  • ,inv [@user] — Check invites of a member.
  • ,inv top — Leaderboard of top inviters.
  • ,inv reset — Reset all invite counts.
  • ,inv reset @user — Reset a specific user’s invites.

⭐ Starboard

Starboard keeps the community’s best content alive by saving starred messages.
  • ,sb set <#channel> — Set starboard channel.
  • ,sb remove — Remove starboard channel.
  • ,sb top — Top starred messages.

⏰ Bump Reminders

Automates Disboard bumps: thanks bumpers, locks channel for cooldown, pings when it’s time again.
  • ,bremind set <#channel> — Set bump reminder channel.
  • ,bremind remove — Disable bump reminders.

🛠️ Utilities

  • ,afk <reason> — Set AFK status.
  • ,quote — Quote a message.
  • ,userinfo — Show user info.
  • ,serverinfo — Show server info.
  • ,welcome set — Setup welcome messages.
  • ,goodbye set — Setup goodbye messages.
